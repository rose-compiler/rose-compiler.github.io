Rose::BinaryAnalysis::CallingConvention Namespace Reference

Description

Support for binary calling conventions.

This namespace has three main classes and a number of auxiliary classes. The main classes are:

  • Definition describes a particular calling convention based on external documentation of that calling convention.
  • Dictionary is a collection of calling convention definitions.
  • Analysis performs an analysis and can determine which definitions best match the characteristics of some specified function.
