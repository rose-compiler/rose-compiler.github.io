SageInterface::Ada::ScopePath Struct Reference

Description

stores a path from an innermost scope to the global scope (not part of the path) in form of a sequence of Sage nodes that represent scopes (SgScopeStatements or SgDeclarationStatements).

  • The path is traversed using the range [rbegin(), rend()) to get the scopes in order from outermost scope to innermost scope.
  • The path may contain scopes without names. Those will be skipped when the qualified name is stringified.
