Rose::BinaryAnalysis::Partitioner2::ModulesLinux::LibcStartMain Class Reference

Description

Basic block callback to add "main" address as a function.

If the last instruction of a basic block is a call to __libc_start_main in a shared library, then one of its arguments is the address of the C "main" function.

Callback method.

This is the method invoked for the callback. The chain argument is the return value from the previous callback in the list (true for the first callback). The successor callbacks use chain to indicate whether subsequent callbacks should do anything.
