Rose::BinaryAnalysis::Partitioner2::ModulesLinux::SyscallSuccessors Class Reference

Description

Basic block callback to detect system calls that don't return.

Examines the instructions of a basic block to determine if they end with a system call that doesn't return, such as Linux's "exit".

Callback method.

This is the method invoked for the callback. The chain argument is the return value from the previous callback in the list (true for the first callback). The successor callbacks use chain to indicate whether subsequent callbacks should do anything.
