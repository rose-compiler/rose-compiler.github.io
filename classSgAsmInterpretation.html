SgAsmInterpretation Class Reference

Description

Represents an interpretation of a binary container.

An interpretation is a collection of the parts of a binary specimen that represent a coherent program, library, core dump, etc. For instance, a Windows PE executable has a DOS interpretation and a Windows interpretation–really two executables in the one container. All the DOS-related stuff will be under one SgAsmInterpretation AST and all the Windows-related stuff will be under another SgAsmInterpretation AST.
